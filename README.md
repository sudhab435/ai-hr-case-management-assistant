# AI-Powered HR Case Management Assistant

## Overview
This project simulates an AI-driven HR case management solution inspired by ServiceNow HR Service Delivery (HRSD). It is designed to demonstrate how AI, workflow automation, and case routing can improve employee support experiences by reducing manual effort and accelerating issue resolution.

The solution uses a virtual HR assistant to answer common employee questions, create HR cases when needed, route requests to the right team, and provide reporting on case trends and service performance.

---

## Business Problem
HR teams often spend significant time responding to repetitive employee questions related to payroll, leave, benefits, onboarding, and policy clarification. This creates delays, increases manual workload, and limits the team's ability to focus on more strategic issues.

This project addresses that problem by simulating:
- AI-powered employee self-service
- Automated HR case intake
- Intelligent case routing
- KPI dashboard reporting

## Solution Architecture
1. Employee submits an HR question
2. AI virtual assistant attempts to answer using structured prompts
3. If unresolved, a case is created
4. Case is categorized by issue type
5. Case is routed to the appropriate team
6. Status and SLA-related fields are tracked
7. Dashboard reports case trends and performance metrics

---

## Workflow Example
### Scenario:
An employee asks:  
**"My leave balance looks incorrect. Can you help?"**

### Process:
1. Employee submits question to virtual assistant
2. Assistant identifies issue as Leave Management
3. Assistant provides a general response
4. If issue requires investigation, a case is created
5. Case is routed to HR Operations
6. Team reviews and updates the case
7. Dashboard reflects updated case activity and turnaround time
