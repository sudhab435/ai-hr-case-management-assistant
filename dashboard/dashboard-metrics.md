# Dashboard Metrics

This dashboard is designed to provide visibility into HR case operations.

## Metrics Included
- Total number of cases
- Number of open cases
- Number of closed cases
- Cases by issue type
- Cases by priority
- Average resolution time
- Assigned team workload
- Case trend by date

## Insights Supported
- Identify the most common issue categories
- Monitor unresolved workload
- Spot high-priority case trends
- Evaluate service efficiency
- Support process improvement decisions
