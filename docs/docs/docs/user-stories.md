# User Stories

## Employee
As an employee, I want to ask HR questions through a virtual assistant so that I can get answers quickly without waiting for manual support.

## HR Agent
As an HR team member, I want employee issues to be categorized and routed correctly so that I can resolve them efficiently.

## HR Manager
As an HR manager, I want to see case trends and performance metrics so that I can improve service delivery.

## Business Analyst
As a business analyst, I want to define workflow logic and reporting requirements so that the HR support process is scalable and measurable.
